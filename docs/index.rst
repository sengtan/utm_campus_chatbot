System Overview
===============

The UTM Campus Assistant Chatbot is a Flask-based web application designed to streamline campus facility management through an intelligent AI-powered interface. The system serves both students and administrators with distinct functionality tailored to their roles.

**Key Features:**
- Facility issue reporting and tracking
- Facility information management and discovery
- Facility booking system with approval workflow
- AI-powered chatbot for natural language queries
- Role-based access control (Student/Admin)
- Real-time dashboard and analytics
