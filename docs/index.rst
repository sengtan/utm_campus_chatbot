About This Manual
-----------------

This manual covers all aspects of administering the UTM Campus Assistant system, including:

- **Installation and Configuration**: Complete setup procedures
- **User Management**: Creating and managing user accounts
- **Facility Management**: Adding, editing, and organizing campus facilities
- **Issue Tracking**: Managing facility issues and maintenance requests
- **Booking System**: Overseeing facility booking requests and schedules
- **System Monitoring**: Keeping track of system health and performance
- **Troubleshooting**: Resolving common problems and issues

System Overview
---------------

The UTM Campus Assistant is a Flask-based web application that provides:

- **AI-Powered Chatbot**: Natural language interface for facility inquiries
- **Issue Reporting System**: Streamlined facility problem reporting
- **Booking Management**: Facility reservation and scheduling system
- **Administrative Dashboard**: Comprehensive management interface
- **Multi-Role Support**: Student and administrator access levels
